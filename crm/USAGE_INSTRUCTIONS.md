# Timetable Interface - Usage Instructions

## For Administrators

### Accessing the Timetable
1. Log in with your admin account
2. Navigate to **សិក្សា (Education)** → **តារាងម៉ោង (Timetable)** in the sidebar
3. The timetable list will display

### Adding a New Timetable Entry

1. Click the **បន្ថែមថ្មី (Add New)** button in the top-right corner
2. Fill in the required information:
   - **Classroom**: Select the class
   - **Subject**: Choose the subject
   - **Teacher**: Assign a teacher (optional)
   - **Time Slot**: Select the day and period
   - **Academic Year**: Select the year
   - **Room**: Enter room number (optional)
3. Click **Save** to create the entry

### Editing an Existing Entry

1. Find the entry in the timetable
2. Click the **✏️ (Pencil/Edit)** button
3. Modify the information
4. Click **Save** to update

### Deleting an Entry

1. Find the entry in the timetable
2. Click the **🗑️ (Trash/Delete)** button
3. Confirm the deletion
4. The entry will be removed

### Using Filters

#### Filter by Classroom
1. Click the **ថ្នាក់រៀន (Classroom)** dropdown
2. Select a specific classroom
3. The timetable will automatically update to show only that class's schedule

#### Filter by Academic Year
1. Click the **ឆ្នាំសិក្សា (Academic Year)** dropdown
2. Select a specific year
3. The timetable will automatically update

#### Clearing Filters
- Select **ថ្នាក់ទាំងអស់ (All Classes)** or **ឆ្នាំទាំងអស់ (All Years)** to reset

### Switching Views

#### List View (បញ្ជី)
- Click the **បញ្ជី** tab
- Best for: Detailed viewing, editing, desktop use
- Shows: All information in a table format

#### Grid View (ក្រឡា)
- Click the **ក្រឡា** tab
- Best for: Quick overview, mobile devices
- Shows: Compact cards organized by day

### Organizing Timetables

#### Best Practices
1. Create time slots first (if not already created)
2. Assign teachers to subjects
3. Create timetable entries for each class
4. Review for conflicts (same teacher, two places)
5. Print or export for distribution

#### Weekly Schedule Setup
For a typical week:
1. Filter by a specific classroom
2. Add entries for Monday (ចន្ទ)
3. Add entries for Tuesday (អង្គារ)
4. Continue through Saturday (សៅរ៍)
5. Review the complete schedule

#### Period Badge
- **វ1, វ2, វ3...** = Period numbers

- Easy to scan

- Format: **07:00 – 07:45**
- Shows start and end time

- Bold text

- Grade and section included

1. Switch to **Grid View (ក្រឡា)** for easier mobile viewing
2. Scroll through days
3. Each card shows one day's schedule
4. Tap cards for more details (if implemented)

### Tips for Teachers

- Check your schedule at the start of each day
- Note any room changes
- Verify time slots before class
- Coordinate with homeroom teachers for conflicts

## For Students

1. Log in with your student account
2. Navigate to **តារាងម៉ោង (Timetable)**
3. Your class schedule will automatically appear

### Understanding Your Schedule

#### What You See
- Only your classroom's timetable
- All subjects for your grade
- Teacher names for each subject
- Time slots for each period

#### Period Information
- **Period Number**: កម្រិត (វ1, វ2, etc.)

- **Teacher**: Who will teach the class

### Preparing for Class

2. Note first and last periods

4. Check room numbers if they change

### Mobile Usage

1. Save the timetable page as a bookmark

3. Screenshot your daily schedule for offline access
4. Check regularly for any updates

## Common Questions

### Q: Why can't I edit the timetable?
**A:** Only administrators have editing rights. Teachers, students, and parents can view only.

### Q: How do I print the timetable?
**A:** Use your browser's print function (Ctrl+P or Cmd+P). The page is print-optimized.

### Q: Can I see multiple classes at once?
**A:** Yes, if you're an admin. Use "All Classes" in the filter. Other users see only their relevant schedule.

### Q: What if there's a conflict?
**A:** Contact the school administrator to resolve scheduling conflicts.

### Q: How often is the timetable updated?
**A:** Changes are immediate. Refresh the page to see the latest schedule.

### Q: Can I export the timetable?
**A:** Use your browser's print function and select "Save as PDF" as the destination.

### Q: What does the period count badge mean?
**A:** It shows how many periods are scheduled for that day.

### Q: Why are some fields empty (—)?
**A:** Some information is optional (like room number or specific teacher). — means "not specified."

## Troubleshooting

### Schedule Not Showing
1. Check if a classroom is selected in filter
2. Try selecting "All Classes"
3. Verify academic year is correct
4. Refresh the page

### Can't Edit Entry
1. Verify you're logged in as admin
2. Check if entry exists
3. Try refreshing the page

### Mobile View Issues
1. Switch to Grid View
2. Rotate device to landscape
3. Zoom out if needed
4. Update your browser

### Print Quality Issues
1. Use browser's print preview
2. Select "Save as PDF" for best quality
3. Adjust margins if needed
4. Use landscape orientation for wide tables
